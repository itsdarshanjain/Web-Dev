Conditions (if-else)
Even/Odd Check: Ek number input lein aur check karein ki wo Even hai ya Odd.

Positive/Negative: Check karein ki ek number Positive hai, Negative hai, ya Zero hai.

Leap Year: Ek saal (year) input lein aur batayein ki wo Leap Year hai ya nahi.

Max of Three: Teen numbers input lein aur batayein ki unme se sabse bada (maximum) kaun sa hai.

Vote Eligibility: Age input lein aur batayein ki person vote dene ke layak hai ya nahi (18+).

2. Loops (for/while)
Print Numbers: 1 se 20 tak saare numbers print karein.

Sum of N Numbers: 1 se 100 tak ke saare numbers ka total (sum) nikaalein.

Table of a Number: User se koi number input lein aur uska multiplication table print karein.

Factorial: Kisi number ka factorial calculate karein (e.g., 5! = 54321 = 120).

Reverse Print: 10 se 1 tak ke numbers ko reverse mein print karein.

3. Switch Case
Days of Week: 1 se 7 tak number input lein aur corresponding din ka naam print karein (1=Monday, etc.).

Months: 1 se 12 tak number input lein aur mahine (month) ka naam print karein.

Vowel/Consonant: Ek character input lein aur check karein ki wo Vowel (a, e, i, o, u) hai ya Consonant.

Calculator (Basic): User se do numbers aur ek operator (+, -, *, /) lein aur result show karein.

Color Codes: Red, Green, Blue input lene par uska full form ya koi specific message print karein.
